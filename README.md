# O-YOGA瑜伽原 — 课程体系与教练手册

面向瑜伽学员和教练的双端内容平台，以结构化课程和体式图解为核心，提供从零基础到高阶的完整练习路径，以及面向教练的口令卡片和教学工具。

## 预览

- **学员端**：[课程体系首页](https://divem.github.io/yoga-manual/)
- **教练端**：[教练教学手册](https://divem.github.io/yoga-manual/instructor/index.html)
- **介绍页**：[给瑜伽馆老板的介绍](https://divem.github.io/yoga-manual/intro.html)

通过 GitHub Pages 部署，推送至 `main` 分支自动发布。

## 教练手册

为瑜伽教练设计的课堂教学工具，基于现有课时数据自动生成，零额外维护。

### 核心功能

- **教学口令卡** — 将课堂序列转为可左右滑动的分步卡片，显示步骤名、时长、引导语
- **课堂时间线** — 按时间比例可视化的节奏条，颜色区分热身/主体/放松阶段
- **体式速查** — SVG 体式图 + 中英梵文名 + 分类标签，快速定位
- **教学要点** — 每节课的对齐要点和安全提示
- **深色模式** — 适合教室昏暗环境，偏好自动保存
- **悬浮导航** — 快捷跳转到时间线/口令/体式/要点/安全区块
- **分类筛选** — 按基础/进阶/工作坊快速过滤

### 技术实现

- 口令卡片使用 CSS `scroll-snap-type: x mandatory` 实现触屏滑动翻页
- 时间线与口令卡片双向联动（点击时间线跳转卡片，翻卡高亮时间线）
- 深色模式通过 CSS 变量 `body.dark` 切换，localStorage 持久化
- 所有页面自包含，无外部依赖

## 本地开发

```bash
# 无需安装依赖，直接启动本地服务器
npx serve .

# 或用 Python
python3 -m http.server 8000
```

## 构建

```bash
node build.js
```

构建脚本会：
1. 读取 `data/` 中的体式和课时数据
2. 生成学员端：13 门课程页 + 85 节课时页
3. 生成教练端：教练手册首页 + 85 节教练课时详情页
4. 渲染 SVG 体式示意图（程序化生成，基于坐标数据）

## 技术特点

- **纯静态** — 零依赖、零构建步骤，发布时不需要 npm
- **自包含 HTML** — 每个页面都是完整的独立 HTML，可离线使用
- **SVG 体式图** — 由 `data/poses.js` 中的坐标数据程序化生成
- **移动优先** — 响应式设计，适配手机 / 平板 / 桌面
- **双端设计** — 学员端与教练端独立页面，各自优化使用体验
- **中文界面** — 全中文内容，面向中文瑜伽社群
