# 流瑜伽 Vinyasa Flow

## 课程定位

Vinyasa 的核心含义是"以特定方式放置"——每个动作都有意识地与呼吸同步。流瑜伽不是"快速做体式"，而是在呼吸的河流中让身体流动。本课程在哈他基础上引入串联（Vinyasa）和编排变化，发展心肺耐力、动态力量和移动中的专注力。

## 适合人群

- 完成哈他瑜伽课程，具备完整拜日式和 40+ 体式基础
- 想提升练习强度、增加动态力量
- 喜欢有节奏感、带有"心流"体验的练习
- 有一定运动基础（跑步、游泳、力量训练）想交叉训练

## 课程目标

1. 掌握标准 Vinyasa 过渡（Chaturanga → 上犬 → 下犬）的安全执行
2. 能在呼吸引导下流畅完成 30-45 分钟的连续序列
3. 理解流瑜伽序列的编排逻辑（高峰体式 / 波浪节奏）
4. 发展 Bandha（收束）在动态练习中的应用
5. 提升心肺耐力与上肢力量

## 课程大纲

### 模块一：Vinyasa 基础（第 1-3 课）

**主题**：呼吸与动作的精确同步

- 一呼一动（one breath one movement）的练习方法
- Chaturanga 四柱支撑的力学分析与训练
  - 常见错误：肩膀低于肘部、腰部塌陷、耸肩
  - 替代：膝盖着地 Chaturanga / 直接到下犬
- 上犬式的肩胛骨位置与腿部激活
- Vinyasa 过渡的三种难度等级

**核心序列**：
- 拜日式 A — 精细化（每个过渡都干净利落）
- 拜日式 B — 加入战士一式的过渡
- 基础 Vinyasa 链接练习

### 模块二：站立流动序列（第 4-6 课）

**主题**：从单侧到双侧的流动编排

- 单侧站立序列的编排（右侧完成再换左侧 vs 左右交替）
- 战士系列的流动变化：战士 I → 战士 II → 侧角 → 半月 的无缝过渡
- 过渡中的平衡训练

**核心序列**：
- 战士流动序列（Warrior Flow）
- 站立平衡序列：战士 III → 半月式 → 站立劈叉
- 扭转流动：扭转侧角 → 扭转半月

### 模块三：核心与手臂支撑（第 7-9 课）

**主题**：建立中心力量

- Bandha 收束法入门：Mula Bandha（根锁）、Uddiyana Bandha（腹锁）
- 核心在流动中的持续参与
- 手臂支撑体式的力量准备

**核心体式**：
- 船式 Navasana 及其动态变体
- 侧板式 Vasisthasana
- 乌鸦式 Bakasana
- 八字扭转式 Astavakrasana（进阶目标）
- 核心强化序列（板式变化系列）

### 模块四：后弯流动（第 10-11 课）

**主题**：在流动中安全后弯

- 后弯体式在 Vinyasa 序列中的位置（序列后半段、身体充分热身后）
- 从桥式到轮式的渐进准备
- 后弯后的反体式序列

**核心序列**：
- 后弯准备流动：蝗虫 → 弓式 → 骆驼 → 桥式/轮式
- 后弯 Vinyasa 链接
- 反体式序列：仰卧抱膝 → 扭转 → 前屈

### 模块五：完整课堂编排（第 12-14 课）

**主题**：理解高峰体式编排法

- 高峰体式（Peak Pose）编排法的逻辑
  - 选定高峰体式 → 分析所需的力量/柔韧/平衡要素 → 在序列中逐步准备
- 波浪节奏：强度递增 → 高峰 → 冷却 → 再递增 → 最终高峰 → 冷却
- 音乐在 Vinyasa 课中的运用

**示例高峰体式编排**：
- 高峰：舞王式 Natarajasana
  - 准备：股四头肌拉伸 → 战士 III（平衡） → 弓式（后弯） → 骆驼式 → 组合
- 高峰：鸟王到战士 III 的连续转换
  - 准备：单腿平衡系列 → 髋关节动态热身 → 核心激活

## 课程结构（单节课 60 分钟）

| 阶段 | 时长 | 内容 |
|------|------|------|
| 开场调息 | 3min | 坐姿调息，建立乌加依呼吸 |
| 热身 | 7min | 猫牛 → 下犬 → 拜日式 A ×3 → 拜日式 B ×2 |
| 站立序列 | 15min | 站立体式的流动串联（左右两侧） |
| 高峰序列 | 15min | 向高峰体式递进的强化序列 |
| 地面序列 | 10min | 坐姿/仰卧体式、扭转、冷却 |
| Savasana | 7min | 挺尸式 |
| 回收 | 3min | 侧卧起身、坐姿回收 |

## 教学要点

### 节奏控制
- 整节课的呼吸节奏应大致统一，避免忽快忽慢
- 教师的口令本身就是节奏器：语速 = 学员的动作速度
- 高峰前适当加快节奏制造强度，冷却时明显放慢

### Chaturanga 安全守则
- 这是 Vinyasa 课中受伤率最高的动作
- 肩膀不低于肘部高度
- 无法标准完成时必须降级（膝盖着地或跳过）
- 一节课中 Chaturanga 次数控制在 10-15 次以内

### 呼吸优先原则
- 如果学员气喘吁吁，说明节奏过快或缺少休息
- 每 10-15 分钟安排一个"重置点"（下犬停留 5 次呼吸）
- 吸气 = 延展/打开动作，呼气 = 折叠/扭转/用力动作

### 音乐建议
- 热身：轻柔有节奏的 ambient/chill
- 高峰段：节奏感更强但不干扰呼吸的音乐
- 冷却/Savasana：纯音乐或自然音效
- 音量始终低于教师口令
