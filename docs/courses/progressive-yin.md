# 阴瑜伽 Yin Yoga

## 课程定位

阴瑜伽由 Paul Grilley 和 Sarah Powers 发展而来，融合了道家经络理论和西方筋膜科学。与"阳性"练习（肌肉收缩、动态流动）互补，阴瑜伽通过长时间的被动停留（3-5 分钟/体式），作用于深层结缔组织——筋膜、韧带和关节囊。

阴瑜伽的三原则：
1. **找到适当的深度**（不是最深的，是"舒适的不适"）
2. **保持静止**（停留期间不调整、不动）
3. **保持时间**（时间是阴瑜伽的"强度"）

## 适合人群

- 有基础的瑜伽练习者，想增加柔韧性和关节活动度
- 高强度运动者（跑步、CrossFit、力量训练）需要恢复和平衡
- 阳性瑜伽练习者（流瑜伽、阿斯汤加）想补充阴性练习
- 对中医经络或筋膜理论感兴趣
- 想在练习中发展冥想和内观能力

## 课程目标

1. 掌握 15+ 个阴瑜伽体式及其变体
2. 理解筋膜系统的基本概念和阴瑜伽的作用机制
3. 了解主要经络与体式的对应关系
4. 学会在长时间停留中运用呼吸和正念技术
5. 能根据自身需求组合 30-45 分钟的阴瑜伽个人练习

## 理论基础

### 筋膜科学
- 筋膜是包裹肌肉、器官、骨骼的连续结缔组织网络
- 筋膜需要长时间、低强度的拉伸才能产生可塑性变化
- 快速拉伸作用于肌肉弹性，慢速停留作用于筋膜可塑性
- 每个人的骨骼结构不同（骨骼形态限制），决定了体式的"天花板"

### 经络理论
- 阴瑜伽中常涉及的六条经络（下肢）：
  - 肾经/膀胱经 → 后侧链（前屈类体式）
  - 肝经/胆经 → 内侧/外侧链（开髋类体式）
  - 脾经/胃经 → 前侧链（后弯/股四头肌拉伸类体式）

## 核心体式库

### 髋部开放

**蝴蝶式 Butterfly**
- 脚掌相对，双膝向两侧打开，脊柱自然前屈
- 目标组织：内收肌群筋膜、髋关节囊
- 经络：肝经、肾经
- 停留：3-5 分钟
- 变体：脚离身体远（更多脊柱前屈）/ 脚靠近身体（更多髋外旋）

**蜻蜓式 Dragonfly**
- 坐姿双腿向两侧打开，脊柱前屈
- 目标组织：内收肌群、腘绳肌内侧
- 经络：肝经、肾经、脾经
- 停留：3-5 分钟
- 变体：前屈 / 侧弯向一条腿 / 抱枕支撑

**天鹅式 Swan / 睡天鹅 Sleeping Swan**
- 类似鸽子式，前腿外旋后腿后伸
- 目标组织：前腿臀部外旋肌、后腿髂腰肌
- 经络：胆经、胃经、肝经
- 停留：3-5 分钟/侧
- 变体：直立天鹅（更多后弯）/ 睡天鹅（前屈折叠）

**方块式 Square / 火烧木式 Fire Log**
- 双小腿上下叠放，脊柱前屈
- 目标组织：臀部深层外旋肌（梨状肌）
- 停留：3-5 分钟/侧
- 变体：上方腿放不到位时用砖支撑膝盖

### 脊柱与后侧链

**毛毛虫式 Caterpillar**
- 坐姿双腿前伸，脊柱自然前屈（允许弓背）
- 与哈他前屈的区别：阴瑜伽中不追求脊柱延展，允许自然弯曲
- 目标组织：脊柱后侧韧带、腘绳肌筋膜
- 经络：膀胱经
- 停留：3-5 分钟

**香蕉式 Banana**
- 仰卧侧弯，身体呈弧形
- 目标组织：体侧筋膜（胸腰筋膜侧链）
- 经络：胆经
- 停留：3-4 分钟/侧

**扭转系列 Twisted Roots / Supine Twist**
- 仰卧扭转，双膝倒向一侧
- 目标组织：脊柱旋转肌群、胸腰筋膜
- 经络：胆经、膀胱经
- 停留：3-5 分钟/侧

### 前侧链与股四头肌

**鞍式 Saddle**
- 跪坐后仰（金刚坐后弯）
- 目标组织：股四头肌、髂腰肌、脊柱前纵韧带
- 经络：胃经、脾经
- 停留：3-5 分钟
- 变体：单腿鞍式（另一腿前伸）/ 抱枕支撑后背
- 注意：膝盖有问题的学员用替代体式

**龙式 Dragon**
- 低弓步，后膝着地
- 目标组织：后腿髂腰肌、股四头肌
- 经络：胃经、肝经
- 停留：3-5 分钟/侧
- 变体：低龙（双手在前脚内侧）/ 扭转龙 / 翼龙（手肘着地）

## 课程结构（单节课 75 分钟）

| 阶段 | 时长 | 内容 |
|------|------|------|
| 开场 | 5min | 仰卧调息，介绍阴瑜伽三原则 |
| 体式序列 | 60min | 8-10 个阴瑜伽体式（含反弹时间） |
| Savasana | 7min | 完全放松 |
| 回收 | 3min | 缓慢唤醒、坐姿结束 |

1. 蝴蝶式 — 5min
2. （反弹：仰卧 1min）
3. 半天鹅式右侧 — 4min
4. 半天鹅式左侧 — 4min
5. （反弹：猫牛式动态 1min）
6. 蜻蜓式 — 5min
7. （反弹：轻摇双腿 1min）
8. 方块式右侧 — 4min
9. 方块式左侧 — 4min

11. 鞍式 — 4min（反向打开前侧）
12. 仰卧扭转 — 4min/侧
13. Savasana — 7min

## 教学要点

### "反弹"（Rebound）
- 每个阴瑜伽体式退出后，仰卧或中立位停留 1-2 分钟
- 这是体式效果"渗透"的时间，不可跳过
- 反弹时可能感到气血流动、麻酥、温热——都是正常反应

### 阴瑜伽 vs 拉伸的区别
- 拉伸目标是肌肉弹性 → 需要温热的肌肉、不宜太久
- 阴瑜伽目标是筋膜可塑性 → 肌肉放松状态下、需要时间
- 阴瑜伽中不追求"更深"——到达一定深度后放松等待即可

### 个体差异的理解
- 骨骼形态（Skeletal Variation）限制了每个人的最大活动范围
- 两个人的蝴蝶式看起来完全不同是正常的
- 教师需要能区分"紧张"（可以通过练习改善）和"压迫"（骨骼限制，无法改变）
- 口令中避免"你应该感觉到……在这里"——改为"注意你在哪里感觉到拉伸"

### 心理层面
- 长时间停留会引发各种心理反应：无聊、焦躁、情绪涌现
- 这是阴瑜伽的一部分——练习"和不适共处"而不是逃避
- 教师可在停留期间提供正念引导、身体扫描或安静的空间
- 不强制闭眼，不要求特定的情绪状态

### 安全注意
- 疼痛（尤其是关节内的尖锐痛）必须立即退出
- 避免在受伤部位做长时间停留
- 腰椎间盘问题慎做前屈类阴瑜伽体式
- 膝盖问题避免鞍式和深度莲花相关体式
