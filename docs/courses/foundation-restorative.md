# 修复瑜伽 Restorative Yoga

## 课程定位

修复瑜伽不是"简单版瑜伽"，而是一种独立的练习体系。核心理念是通过辅具的完全支撑，让身体在零肌肉参与的状态下进入深度放松。每个体式停留 5-15 分钟，整节课仅做 4-6 个体式。这是一堂关于"放下努力"的课。

## 适合人群

- 完全零基础——无需任何瑜伽经验
- 工作压力大、交感神经长期亢奋
- 失眠或睡眠质量差
- 运动后恢复、身体疲劳
- 伤病恢复期（需告知老师具体情况）
- 经期不适
- 作为高强度瑜伽练习的补充

## 课程目标

1. 体验并理解副交感神经主导的深度放松状态
2. 学会使用辅具搭建 8-10 个修复体式
3. 掌握身体扫描和呼吸放松技术
4. 改善睡眠质量和压力管理能力
5. 建立"不做什么"也是练习的认知

## 核心体式库

### 仰卧系列

**支撑仰卧束角式 Supta Baddha Konasana**
- 辅具：抱枕纵放于脊柱下方，两侧膝盖下各一块砖/毛毯
- 功效：打开胸腔、髋部，舒缓经期不适
- 停留：8-15 分钟
- 调整：肩膀下方垫毛毯角，减少肩部紧张

**支撑桥式 Supported Setu Bandhasana**
- 辅具：骶骨下放砖（最低高度）或抱枕
- 功效：温和后弯，缓解腰部疲劳
- 停留：5-10 分钟
- 调整：膝盖内侧放砖保持平行，或用带子绑膝盖

**仰卧抬腿靠墙 Viparita Karani**
- 辅具：臀部下方放抱枕/折叠毛毯，双腿靠墙
- 功效：缓解腿部水肿和疲劳，温和倒置
- 停留：10-15 分钟
- 调整：腿可以分开呈 V 形，或脚掌相对成束角

### 前屈系列

**支撑婴儿式 Supported Balasana**
- 辅具：身体前方放抱枕，额头或侧脸落在抱枕上
- 功效：安抚神经系统，缓解背部紧张
- 停留：5-8 分钟（每侧转头各一半时间）
- 调整：膝盖之间可放毛毯，减少膝盖压力

**支撑坐姿前屈 Supported Paschimottanasana**
- 辅具：大腿上方放抱枕，额头落在抱枕上
- 功效：安静的内省体式，拉伸背侧链
- 停留：5-10 分钟
- 调整：坐骨下方垫毛毯抬高骨盆

### 侧卧与扭转系列

**支撑侧卧放松 Supported Side-Lying Savasana**
- 辅具：抱枕放在体侧，上方腿搭在抱枕上
- 功效：适合孕晚期或仰卧不适者的替代放松姿势
- 停留：5-8 分钟

**支撑仰卧扭转 Supported Supta Matsyendrasana**
- 辅具：身体侧面放抱枕，膝盖之间放砖或毛毯
- 功效：温和扭转，释放脊柱张力
- 停留：5-8 分钟/侧
- 调整：扭转幅度由辅具高度控制，完全不需要肌肉用力

## 课程结构（单节课 60 分钟）

| 阶段 | 时长 | 内容 |
|------|------|------|
| 开始仪式 | 5min | 仰卧位抵达，身体扫描，设定放松意图 |
| 体式一 | 10min | 通常为仰卧打开类（束角式或靠墙抬腿） |
| 体式二 | 10min | 前屈或侧弯类 |
| 体式三 | 10min | 扭转类 |
| 体式四 | 10min | 支撑后弯或另一仰卧体式 |
| 挺尸式 | 10min | 完全覆盖毛毯的 Savasana |
| 唤醒回收 | 5min | 缓慢唤醒、侧卧过渡、坐姿结束 |

## 教学要点

### 环境营造
- 灯光调暗或使用烛光
- 室温略高于常规课（24-26°C），因为学员不运动、体温会下降
- 准备眼枕，Savasana 时使用
- 可配合轻柔背景音乐或颂钵

### 辅具搭建
- 课前将所有辅具准备到每位学员垫子旁
- 每个体式进入前，先完整演示辅具摆放方式
- 辅具的核心原则：**身体不应有任何部位悬空**
- 每位学员的辅具配置可能不同，需逐一检查

### 引导风格
- 语速慢于正常教学，音量略低
- 进入体式后，减少指令，给予长段安静时间
- 可穿插身体扫描引导（"注意你的肩膀是否还在用力……让它放下"）
- 避免突然的指令或动作提示

### 特殊人群注意
- **孕期**：避免仰卧体式（16周后），替换为侧卧变体
- **腰椎问题**：桥式辅具高度调低，避免过度前凸
- **焦虑/PTSD**：有些人无法闭眼或在安静中感到不安，允许睁眼、提供额外毛毯（安全感）

## 推荐练习频率

- 日常修复：每周 1-2 次
- 高压力期：每天 1 个修复体式（靠墙抬腿或支撑束角式），15 分钟
- 作为补充：高强度练习日的次日安排一次修复课
