# 哈他瑜伽 Hatha Yoga

## 课程定位

哈他瑜伽是所有体式练习的母体。本课程注重每个体式的精准停留，通过较长的保持时间（5-8 次呼吸）深入体会身体在体式中的工作方式。节奏缓慢但绝不轻松——静态保持对肌肉耐力和专注力的要求比流动练习更高。

## 适合人群

- 完成瑜伽入门课程，具备 15 个基础体式的练习经验
- 想要深化体式理解，不满足于"跟着做"而想知道"为什么这样做"
- 身体有轻度不适（肩颈紧张、腰部酸痛），需要针对性的体式调整
- 不喜欢快节奏流动，偏好沉稳、深入的练习风格

## 课程目标

1. 将体式库扩展至 40+ 个体式，涵盖站立、坐姿、扭转、后弯、倒置全类型
2. 理解每个体式的骨骼对齐原则（alignment），能自我检查和调整
3. 掌握乌加依呼吸在体式中的持续运用
4. 建立体式的"内在觉知"——从外在形状转向内在感受
5. 初步理解体式的功效与禁忌，具备自我保护意识

## 课程大纲

### 模块一：站立体式深化（第 1-3 课）

**主题**：从根基到延展

- 站姿体式的精细对齐：骨盆中正、肋骨内收、肩胛下沉
- 三组对立力量的理解：向下扎根 vs 向上延展
- 辅具在站姿中的进阶用法

**核心体式**：
- 战士系列 I / II / III 深化
- 半月式 Ardha Chandrasana（砖辅助）
- 扭转侧角式 Parivrtta Parsvakonasana
- 加强侧伸展式 Parsvottanasana
- 鹰式 Garudasana

### 模块二：前屈与髋关节（第 4-6 课）

**主题**：骨盆前倾的艺术

- 前屈的核心原则：从髋关节折叠而非腰椎弯曲
- 腘绳肌 vs 髋屈肌的区分感知
- 坐骨的觉知与骨盆倾斜方向

**核心体式**：
- 坐姿前屈 Paschimottanasana（带子辅助 → 独立）
- 束角式 Baddha Konasana（砖支撑膝盖）
- 鸽子式 Eka Pada Rajakapotasana（前屈变体）
- 双角式 Prasarita Padottanasana
- 牛面式 Gomukhasana

### 模块三：后弯与胸椎灵活性（第 7-9 课）

**主题**：安全地打开前侧身体

- 后弯的准备原则：先延展再弯曲、胸椎主导而非腰椎
- 肩胛骨收拢与胸腔打开的配合
- 后弯中的核心保护

**核心体式**：
- 蝗虫式 Salabhasana
- 眼镜蛇式 → 上犬式过渡 Bhujangasana → Urdhva Mukha Svanasana
- 弓式 Dhanurasana
- 桥式 Setu Bandhasana
- 骆驼式 Ustrasana（手扶腰变体 → 完整）

### 模块四：扭转与脊柱健康（第 10-11 课）

**主题**：从中轴线出发的旋转

- 扭转的三条原则：先延展、从腹部启动、呼气加深
- 坐姿扭转 vs 站姿扭转的力学差异
- 扭转对消化系统和脊柱健康的益处

**核心体式**：
- 脊柱扭转 Ardha Matsyendrasana
- 仰卧扭转 Supta Matsyendrasana
- 扭转三角式 Parivrtta Trikonasana
- 圣哲玛里奇式 Marichyasana I / III

### 模块五：倒置与平衡（第 12-13 课）

**主题**：翻转视角

- 倒置体式的生理益处与禁忌
- 肩倒立的安全准备（毛毯辅助、颈椎保护）
- 平衡体式中的凝视点（Drishti）与心理稳定

**核心体式**：
- 双角式倒立变体（头低于心脏的安全过渡）
- 肩倒立 Sarvangasana（毛毯支撑）
- 犁式 Halasana
- 乌鸦式预备 Bakasana prep（砖辅助）
- 头倒立预备 Sirsasana prep（靠墙）

### 模块六：整合序列（第 14-16 课）

**主题**：将体式编织为完整练习

- 经典哈他序列的编排逻辑：热身→站立→坐姿→后弯→倒置→放松
- 反体式（counter pose）的概念与应用
- 完整 75 分钟哈他序列的体验

## 课程结构（单节课 75 分钟）

| 阶段 | 时长 | 内容 |
|------|------|------|
| 调息冥想 | 5min | 坐姿调息，乌加依呼吸建立 |
| 热身 | 10min | 拜日式 A × 3 + 拜日式 B × 2 |
| 站立序列 | 15min | 当堂主题相关的站立体式 |
| 主题体式 | 25min | 核心教学——分解、调整、停留 |
| 整合与反体式 | 10min | 反向练习、缓冲序列 |
| 放松收束 | 10min | 仰卧扭转 → 挺尸式 → 坐姿回收 |

## 教学要点

### 辅助调整（Adjustments）
- 先口头提示，再目视确认，最后征得同意后手法辅助
- 调整目的是"引导方向"而非"推到位"
- 高频调整区域：骨盆方向、肩胛骨位置、膝盖对齐

### 体式停留引导
- 每个体式保持 5-8 次呼吸
- 停留期间的引导节奏：进入 → 粗调 → 细调 → 觉知内在 → 退出
- 避免停留期间的长时间沉默，也避免过度指令

### 序列编排原则
- 每节课有一个明确的身体主题（如：开髋 / 后弯准备）
- 体式排列遵循"简单→复杂→高峰体式→冷却"的弧线
- 高峰体式不应出现在课程的最后三分之一

## 结业标准

完成 16 次课程后，学员应能够：
- [ ] 独立完成完整拜日式 A 和 B
- [ ] 在 40 个体式中保持正确对齐与稳定呼吸
- [ ] 理解主要体式的禁忌症（至少说出 10 个体式的禁忌）
- [ ] 能根据自身状况选择体式变体
- [ ] 具备进入流瑜伽、阴瑜伽或艾扬格课程的能力
