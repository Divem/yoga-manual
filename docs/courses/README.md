# 课程体系总览

本平台课程体系分为三大类别，覆盖从零基础到高阶教学的完整路径。

## 课程结构

### 一、基础课程（Foundation）

适合零基础或初学者，建立正确的练习习惯和身体意识。

| 课程 | 难度 | 时长 | 说明 |
|------|------|------|------|
| [瑜伽入门](./foundation-yoga-basics.md) | ⭐ | 60min | 零基础第一课，建立基本体式认知 |
| [哈他瑜伽](./foundation-hatha.md) | ⭐⭐ | 75min | 经典体式精讲，注重对齐与停留 |
| [修复瑜伽](./foundation-restorative.md) | ⭐ | 60min | 辅具支撑下的深度放松与修复 |

### 二、进阶课程（Progressive）

在基础之上发展力量、柔韧性和专注力，探索不同流派风格。

| 课程 | 难度 | 时长 | 说明 |
|------|------|------|------|
| [流瑜伽](./progressive-vinyasa.md) | ⭐⭐⭐ | 60min | 呼吸引导的流动序列 |
| [阿斯汤加](./progressive-ashtanga.md) | ⭐⭐⭐⭐ | 90min | 传统固定序列，Mysore 与 Led 双模式 |
| [艾扬格精准瑜伽](./progressive-iyengar.md) | ⭐⭐⭐ | 90min | 辅具辅助，极致精准对齐 |
| [阴瑜伽](./progressive-yin.md) | ⭐⭐ | 75min | 长时间停留，深入筋膜与经络 |

### 三、专题工作坊（Workshop）

针对特定目标的主题深入课程，可独立参加也可作为常规课程的补充。

| 课程 | 难度 | 时长 | 说明 |
|------|------|------|------|
| [开髋专题](./workshop-hip-opening.md) | ⭐⭐~⭐⭐⭐ | 120min | 髋关节灵活性与稳定性的系统训练 |
| [后弯专题](./workshop-backbend.md) | ⭐⭐⭐~⭐⭐⭐⭐ | 120min | 安全打开胸椎与肩部的后弯进阶 |
| [倒立专题](./workshop-inversion.md) | ⭐⭐⭐⭐ | 120min | 从基础到自由倒立的系统训练 |
| [冥想与呼吸法](./workshop-meditation-pranayama.md) | ⭐ | 60min | 呼吸控制与冥想的系统入门 |
| [孕产瑜伽](./workshop-prenatal.md) | ⭐⭐ | 60min | 孕期各阶段的安全练习方案 |
| [脊柱理疗](./workshop-spine-therapy.md) | ⭐⭐ | 75min | 针对久坐人群的脊柱健康课程 |

## 难度说明

- ⭐ 零基础可参加
- ⭐⭐ 需3个月以上练习基础
- ⭐⭐⭐ 需1年以上规律练习
- ⭐⭐⭐⭐ 需2年以上且具备相应体式基础

## 课程推荐路径

```
零基础 → 瑜伽入门 → 哈他瑜伽 ─┬→ 流瑜伽 → 阿斯汤加
                              ├→ 艾扬格精准瑜伽
                              ├→ 阴瑜伽
                              └→ 各专题工作坊

需要放松 → 修复瑜伽 / 阴瑜伽
身体不适 → 脊柱理疗 / 修复瑜伽
孕期妈妈 → 孕产瑜伽
想深入内在 → 冥想与呼吸法
```
