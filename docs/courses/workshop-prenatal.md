# 孕产瑜伽 Prenatal Yoga

## 工作坊定位

孕期身体的每一天都在变化，而瑜伽可以为这段旅程提供身体上的缓解、心理上的安定，以及为分娩和产后恢复做准备的力量。本工作坊面向孕中晚期（14 周后）的准妈妈，提供安全、温和、有针对性的练习方案。

## 重要声明

**本工作坊不替代医疗建议。** 所有学员必须：
- 获得主治医生的书面许可
- 无高危妊娠因素（前置胎盘、妊娠高血压、先兆子痫等）
- 如有妊娠糖尿病、多胎妊娠等特殊情况，需单独评估

### 三个孕期的一般性指导

| 孕期 | 周数 | 身体状态 | 练习重点 |
|------|------|---------|---------|
| 第一孕期 | 0-13周 | 激素剧烈变化、疲劳、恶心 | 休息为主、仅做温和呼吸和冥想。此阶段流产率最高，不建议参加瑜伽课程 |
| 第二孕期 | 14-27周 | 能量恢复、腹部开始隆起、重心变化 | 适度练习、力量建设、疼痛缓解 |
| 第三孕期 | 28-40周 | 腹部明显、腰背压力大、耻骨联合痛 | 姿势矫正、分娩准备呼吸、温和运动 |

## 孕期瑜伽的核心原则

### 绝对禁忌
- **仰卧**：孕 20 周后避免长时间仰卧（子宫压迫下腔静脉 → 血压下降、头晕）
- **俯卧**：腹部不可承压
- **深扭转**：腹部扭转方向（顺时针扭转可以）
- **深度后弯**：过度拉伸腹部前侧
- **倒置**：头倒立、肩倒立等（孕早期后禁做）
- **屏气/激烈呼吸**：胎儿需要持续供氧
- **跳跃/剧烈转换**：重心不稳定，有摔倒风险
- **腹直肌过度使用**：避免仰卧起坐、船式等强烈核心收缩
- **腹部着地/承压**：任何直接挤压腹部的动作

### 核心练习原则
- **呼吸是第一位**：永远保持顺畅呼吸，不憋气
- **膝盖始终微屈**：松弛素（relaxin）使韧带松弛，关节不稳定
- **重心前移**：适应身体重心的变化，避免摔倒
- **降温练习**：核心体温升高对胎儿有风险——避免高温瑜伽
- **倾听身体**：任何不适立即停止

## 课程结构（单节课 60 分钟）

| 阶段 | 时长 | 内容 |
|------|------|------|
| 围坐分享 | 5min | 每位妈妈简述当周身体感受 |
| 热身 | 10min | 关节活动、骨盆摆动、温和拉伸 |
| 站立序列 | 15min | 安全的站立体式，注重骨盆稳定 |
| 坐姿/侧卧序列 | 15min | 侧卧替代仰卧，开髋与呼吸 |
| 分娩准备 | 10min | 呼吸法 + 会阴保护练习 |
| 放松 | 5min | 侧卧 Savasana |

## 核心体式库

### 站立序列

**战士二式 Virabhadrasana II（孕期改良版）**
- 双脚距离缩短（重心更稳）
- 手叉腰或轻放大腿（代替手臂完全伸展，节省体力）
- 膝盖不超过脚尖
- 功效：打开髋部、增强腿部力量

**侧角伸展式 Utthita Parsvakonasana（孕期改良版）**
- 前手肘支撑在大腿上（代替手触地面）
- 上方手臂自然搭肩/叉腰
- 功效：侧腰拉伸、缓解肋骨区域的压力

**女神式 Goddess Pose**
- 双脚宽于髋，脚尖外展 45°
- 深蹲，膝盖不超过脚尖
- 双手在胸前合十或自然下垂
- 功效：加强骨盆底肌、打开髋部、为分娩体位做准备

### 坐姿序列

**侧卧束角式 Side-Lying Baddha Konasana**
- 左侧卧，右腿弯曲搭在抱枕/毛毯上
- 毛毯或抱枕支撑头部和上身
- 功效：温和开髋，完全避免仰卧

**坐角式 Upavistha Konasana（温和版）**
- 坐姿双腿打开，脊柱自然直立（不强求前屈）
- 手臂轻放在身后的地面上支撑
- 功效：内收肌拉伸、为分娩开髋

**猫牛式 Marjaryasana-Bitilakasana**
- 四足跪姿，温和起伏
- 牛式时允许腹部下垂（不挤压）
- 功效：缓解腰背酸痛、协调呼吸

### 分娩准备

**骨盆底肌觉知**
- 凯格尔运动的瑜伽版本
- 练习方式：吸气放松，呼气时想象电梯门关闭——从一层到四层逐层提起
- 重要性：分娩时需要主动放松骨盆底肌，平时需要增强它的力量
- 注意：收缩后必须完全放松——只收缩不会放松会让分娩更困难

**深蹲呼吸法（第一产程呼吸）**
- 深蹲姿势（背后靠球或椅支撑）
- 吸气 4 秒，呼气 6-8 秒
- 呼气时嘴唇放松（像吹蜡烛但轻柔）
- 功效：放松会阴、应对宫缩

**哼鸣式呼吸（第二产程呼吸）**
- 嘴唇振动发声呼气（"呼——"的声音）
- 吸气短促，呼气绵长
- 功效：帮助在宫缩时保持放松和节奏

## 孕期常见不适与体式处方

| 不适症状 | 推荐体式 | 避免 |
|---------|---------|------|
| 腰背酸痛 | 猫牛式、骨盆摆动、四足跪姿 | 深度前屈、深后弯 |
| 耻骨联合痛 | 侧卧束角式、仰卧蝴蝶（短时间）| 双腿过度分开 |
| 下肢水肿 | 侧卧抬腿、脚踝环绕 | 长时间站立 |
| 失眠 | 侧卧 Savasana + 身体扫描冥想 | 任何刺激性练习 |
| 消化不良 | 坐姿扭转（非腹部方向）、坐角式 | 俯卧、深度前屈 |
| 情绪波动 | 呼吸法、正念冥想、小组分享 | 过于刺激的练习 |
| 气短 | 上胸腔呼吸觉知、靠墙坐姿 | 俯卧或腹部承压 |

## 产后恢复指引

### 产后 6 周内
- 仅做呼吸练习和温和盆底肌恢复
- 无腹部强烈收缩（等待腹直肌分离检查结果）
- 避免任何核心力量练习

### 产后 6 周后（医生确认后）
- 从修复瑜伽重新开始
- 重点关注：盆底肌恢复、腹横肌重新激活、肩颈缓解（抱娃导致）
- 避免高强度练习至少 3 个月

### 产后常见问题
- **腹直肌分离**：需先恢复腹直肌再进行核心练习
- **盆底肌松弛**：凯格尔运动逐步加强
- **手腕疼痛（妈妈手）**：鹰式手臂拉伸、手腕伸展

## 教学要点

### 安全底线
- 每节课前确认每位学员的孕周和医生许可状态
- 如有出血、头痛、视力模糊、肿胀——建议立即就医
- 不做任何按摩手法（尤其是腹部和特定穴位）
- 教室内地面必须防滑、温度舒适

### 情感支持
- 产前焦虑非常普遍——给予倾听和接纳
- 课上可留出简短的交流时间（围坐分享）
- 不给出医疗建议——遇到具体医学问题建议咨询医生
- 肯定每位准妈妈的身体感受和选择

### 班级氛围
- 小班教学（12 人以内）
- 营造安全、放松、互助的氛围
- 不比较身体状态——每个人的孕期体验都不同
