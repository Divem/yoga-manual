## 字体

| Token | 值 | 用途 |
|-------|-----|------|
| `--font-d` | `'Fraunces', Georgia, 'Noto Serif SC', serif` | 装饰字体，标题、品牌、强调 |
| `--font-b` | `-apple-system, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', 'Noto Sans SC', sans-serif` | 正文字体，中文优先系统字体 |
| `--font-u` | `'Work Sans', -apple-system, 'PingFang SC', sans-serif` | UI 字体，标签、按钮、导航 |

## 间距

| Token | 值 | 用途 |
|-------|-----|------|
| `--nav-h` | `56px` | 导航栏高度 |
| `--safe-b` | `env(safe-area-inset-bottom, 0px)` | 底部安全区域（iPhone 刘海屏） |

### 通用间距约定

- 页面内边距：`16-24px`
- 内容区最大宽度：`680px`（课程详情）、`720px`（教练手册）、`900px`（通用）
- 区块间距：`32px`
- 卡片内边距：`20px`
- 元素间距：`8px`（紧凑）、`12px`（标准）、`16px`（宽松）

## SVG 体式图

| 属性 | 值 | 说明 |
|------|-----|------|
| 画布 | 200 x 280 | 体式坐标系统 |
| 描边色 | `--primary` (`#3A5248`) | 骨架线条颜色，可按课程分类色覆盖 |

## 动效

| 用途 | 值 | 说明 |
|------|-----|------|
| 呼吸动画 | `12s ease-in-out infinite` | Hero 区域呼吸圆环 |
| 滚动提示 | `2s ease-in-out infinite` | 向下箭头脉冲 |
| 渐入动画 | `.8s ease-out` | fade-up（从下方淡入） |
| 过渡 | `.25s` | 按钮、卡片、标签 |
| 噪点纹理 | `opacity: .028` | 全局覆盖的 SVG 噪点 |
| 背景模糊 | `blur(16px)` | 导航栏毛玻璃 |
